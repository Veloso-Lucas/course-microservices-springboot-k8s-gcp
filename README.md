# Course Microservices Spring Boot K8s GCP

This repository contains the projects, source code, notes, and practical exercises developed during a microservices course focused on Spring Boot, Docker, Kubernetes, and Google Cloud Platform.

The main goal of this repository is to study and practice modern backend architecture concepts, including distributed systems, containerization, orchestration, cloud deployment, and service communication.

## About

This project is part of my learning journey in microservices architecture and cloud-native development.

Throughout this course, the focus is on building backend services using Spring Boot and applying infrastructure and deployment concepts with Docker, Kubernetes, and Google Cloud Platform.

## Main Topics

- Microservices architecture
- Spring Boot applications
- REST APIs
- Service communication
- Docker containers
- Kubernetes orchestration
- Google Cloud Platform deployment
- Cloud-native backend development
- Distributed systems concepts
- Scalable application architecture

## Purpose

The purpose of this repository is to centralize all course-related code and studies in one place, making it easier to track progress, review concepts, and document my evolution with microservices and cloud technologies.

## Status

This repository is intended for study and practice purposes.

It may evolve over time as new modules, exercises, and improvements are added during the course.
